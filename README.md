A sleek, single-page web application designed to demystify the complex tax structure of new vehicles in Turkey. This tool provides a detailed breakdown of a vehicle's final price, reverse-calculating the base cost from the turnkey price and clearly visualizing the amounts for √ñTV (Special Consumption Tax) and KDV (VAT).

### ‚ú® Key Features

-   **Detailed Tax Calculation:** Accurately reverse-calculates the tax-free base price, √ñTV amount, and KDV (VAT) amount from the final turnkey price.
-   **Multiple Vehicle Types:** Supports different tax brackets for:
    -   Gasoline / Diesel
    -   Hybrid
    -   Electric Vehicles
-   **Dynamic Brackets:** Automatically uses the correct √ñTV brackets based on the vehicle's base price and engine specifications (cylinder capacity for combustion/hybrid, motor power for electric).
-   **Interactive Donut Chart:** Visually represents the distribution of the final price between the vehicle's base cost, √ñTV, and KDV.
-   **Insightful Analysis:**
    -   Calculates how many months of minimum-wage work are required to pay off the total tax amount.
    -   Compares the tax paid for a fossil fuel or hybrid vehicle to a similarly priced electric vehicle, highlighting potential tax savings.
-   **Modern UI:** A clean, dark-themed, and fully responsive interface built with Tailwind CSS.
-   **Showcase Section:** Features a list of the most affordable new cars currently available in the Turkish market.
